Para usar o programa você deve ter o Python, pip e importar o flask no seu computador.
Se não tiver basta instalar com os comandos:
	sudo apt-get install python
	sudo apt-get install python-pip
	pip install flask

Agora, com o python e pip instalados, vá no terminal, dentro da pasta testA55 e digite:
	python app.py

Vá ao navegador e digite 127.0.0.1:5000/devs

Estaram aí os nomes de programadores.

Para ter a lista de apenas programadores python basta digitar 127.0.0.1:5000/devs/python
Ou para um id em especifico 127.0.0.1:5000/devs/1 (para o id 1)
